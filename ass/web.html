    DEFINITION OF TERMS

    1. INTERNET:
      2. Internet is simply defined as the interconnection between computer users.
    2. BROWSER:
      3. Browsers are software applications that renders webpages. Example includes Chrome, Uc, Moxila firefo
    3. CLIENT-MACHINE:
      4. They are computers that are connected to internet.
    4. SERVER:
      5. They are dedicated computers that serves other client machines with information as requested and they are always connected to the internet 24/7.
    5. HOMEPAGE:
      6. It is the very first page that always display when a website is been access. Example includes Dynamic and statics
    6. HTML:
      7. Hypertext markup language is define as coding language that uses markup method to create hypertext. it is also use in structuring webpage and layout.
    7. CSS:
      8. Cascoding stylesheet is simply use to style a webpage.
    8. JAVA SCRIPT:
      9. It is used in adding functionality to webpage.
    9. ATTRIBUTE:
      10. Attribute is used to change default settings or appearance in a tag and it can be used only in opening tags.
    10. TAGS:
      11. Tags are command synta that tell the computer what to do. The types includes Container tag and Empty or self closing tag
    11. TEXT EDITORS
      12. Text editors are use to write and edit source code. Example include Atom, Sublime, Notepad ++
    12. PROTOCOL:
      13. They are set of rules establishes or guidesIntermachine communication.
    13. UNICODE:
      14. They are sets of characters known to the computer.
